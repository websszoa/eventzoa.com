# 칼로리 사이트

### Getting Started

Next.js 프로젝트 생성

```bash
npx create-next-app@latest ./      # Next.js 프로젝트 생성
npm i @supabase/supabase-js        # Supabase 클라이언트
npm i @supabase/ssr                # Supabase SSR 지원
npm i next-themes                  # 다크모드 테마 지원
npx shadcn@latest init             # shadcn/ui 초기화
npx shadcn@latest add button
npx shadcn@latest add sonner
npx shadcn@latest add sheet
npx shadcn@latest add scroll-area
npx shadcn@latest add separator
npx shadcn@latest add form
npx shadcn@latest add textarea
npx shadcn@latest add input
npx shadcn@latest add badge
npx shadcn@latest add dialog
npx shadcn@latest add cheackbox
npx shadcn@latest add sidebar
npx shadcn@latest add input-otp
npx shadcn@latest add table
```
